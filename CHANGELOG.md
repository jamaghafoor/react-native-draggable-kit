# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-01

### Added

- Initial release of `react-native-draggable-kit`.
- `<DraggableList>` component with vertical and horizontal reordering.
- `<DragHandle>` component for handle-only drag mode.
- `useDraggableList` hook for custom UI integration.
- UI-thread gesture and animation handling via Reanimated worklets.
- Auto-scroll when dragging near container edges.
- Variable-height/width item support with automatic measurement.
- Long-press and immediate drag activation modes.
- Configurable drag appearance (scale, opacity, elevation).
- Pluggable haptic feedback callback.
- Accessibility support with Move Up/Down/Top/Bottom actions.
- `reorderArray` utility function.
- TypeScript type definitions.
- Example app with 5 demo screens.
- Comprehensive documentation and API reference.
